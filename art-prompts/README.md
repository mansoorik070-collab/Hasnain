# Free Style Art Prompt Package

This package contains concept-ready prompt files, visual direction notes, and placeholder assets for the **Free Style** universe. It is designed to help concept artists or image-generation workflows produce a consistent 2099 dimensional-war cyber-fantasy style.

## Included

- Logo concepts
- Loading screen concepts
- Main menu background directions
- Fighter prompt pack
- Arena prompt pack
- Secret character prompt pack
- Boss intro art directions
- Unified style guide

## Usage Notes

- Keep silhouettes readable on small Android screens.
- Favor strong value separation and high-contrast focal lighting.
- Preserve gameplay readability over visual complexity.
- Use layered export outputs so UI branding can be repurposed across splash, title, loading, and store assets.
