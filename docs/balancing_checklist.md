# Balancing Checklist

## Combat Balance
- [ ] Light attacks are safe but not overly rewarding without confirms
- [ ] Heavy attacks have punishable recovery windows
- [ ] Special meter gain is consistent across match lengths
- [ ] Overdrive activation is powerful but not auto-win
- [ ] Perfect block reward is meaningful but execution-gated
- [ ] Guard break is viable against passive defense
- [ ] Air juggle loops are capped to prevent infinite combos

## Roster Balance
- [ ] Each fighter has at least one clear strength and weakness
- [ ] Hidden characters do not outclass starter roster by default
- [ ] Boss movesets feel unfair cinematically but fair mechanically
- [ ] Ranked PvP uses normalized stats where intended

## Economy Balance
- [ ] Story rewards unlock content at a satisfying pace
- [ ] Daily rewards support retention without invalidating campaign progression
- [ ] Upgrade costs scale sensibly across early, mid, and end game
- [ ] Honor Token output supports seasonal progression goals

## PvP Ruleset Checks
- [ ] Match length is acceptable on mobile sessions
- [ ] Ladder demotion/promotion pacing feels fair
- [ ] Win streak bonuses do not distort high-rank outcomes
- [ ] Reconnect or rematch flows do not allow exploitation
